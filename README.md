# Human Resource Information System (HRIS)

## Overview
This HRIS is a web-based system designed to help HR staff manage employees, attendance, and announcements efficiently. It includes live attendance tracking through mock biometric functions and supports sending announcements via SMS or email. The system provides an organized, user-friendly workflow with real-time updates using AJAX.

---

## Features

### **Dashboard**
- Displays an overview of HR metrics and employee summaries.
- Shows quick statistics for monitoring workforce activity.

### **Employee Management**
- View all employee information.
- Add, update, or manage employee records.
- Organized table view for quick navigation.

### **Attendance Tracking**
- Live attendance monitoring.
- Uses mock biometric recording (manual clock-in and clock-out buttons).
- Real-time rendering powered by AJAX.

### **Announcements**
- HR can broadcast announcements to staff.
- Supports sending messages via **SMS** or **Email**.
- Keeps a history of outgoing announcements.

### **User Settings**
- Profile page for the logged-in HR user.
- Update account details and personal information.

---

## Technology Stack
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Live Updates:** AJAX
